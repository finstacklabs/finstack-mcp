# Interest To Payment

Use this document when someone says they are interested in FinStack Brief.

## 1. First Reply

```text
Thanks. I’m opening early access for FinStack Brief right now in a manual way.

To set you up, send me:
1. your use case: personal / creator / desk
2. your watchlist symbols
3. preferred delivery: email / Telegram / WhatsApp-style text
4. how often you want it: daily / weekly
5. what matters most: market snapshot / earnings / corporate actions / watchlist summary
```

## 2. If They Reply With Details

```text
Perfect. I’ll use that to prepare a sample brief for you.

What you’ll get first:
- Indian market snapshot
- top movers
- sector context
- watchlist updates
- earnings / corporate actions when relevant
- short written summary

I’ll send a sample first, then we can decide whether to continue with a paid monthly setup.
```

## 3. If They Ask Price

```text
Current early-access pricing is simple:

- Personal brief: INR 499-999/month
- Creator / desk workflow: starts at INR 2,999/month

I’m still doing manual onboarding, so pricing depends on watchlist size, delivery format, and frequency.
```

## 4. If They Ask What Makes It Different

```text
The difference is that this is not just raw market data or another generic finance tool.

It is an India-first daily brief workflow built on top of a live MCP market-data engine, focused on turning market state, watchlist events, earnings, and corporate actions into something usable every day.
```

## 5. If They Say “Send Sample”

```text
Sure. Send me:
- watchlist symbols
- preferred format: email / Telegram / WhatsApp-style
- personal / creator / desk use case

I’ll prepare a sample brief from that.
```

## 6. Onboarding Questions

Ask this if you want a cleaner structure:

```text
Quick onboarding:

1. Name:
2. Use case: personal / creator / desk
3. Watchlist:
4. Delivery format: email / Telegram / WhatsApp-style
5. Frequency: daily / weekly
6. Time preference: before market / after market
7. Most important sections:
8. Anything you want excluded:
```

## 7. After Sending Sample

```text
I’ve sent the sample brief.

Tell me honestly:
1. what was useful
2. what felt unnecessary
3. what format you prefer
4. whether you would want this daily

If it feels valuable, I can set you up on an early monthly plan.
```

## 8. Soft Close

```text
If you want, I can start you on early access this month.

The manual setup keeps it flexible while I refine the workflow around real users instead of guessing.
```

## 9. Payment Confirmation

```text
Great. I’ll confirm your setup with:
- watchlist
- delivery format
- delivery schedule
- monthly price

Once confirmed, I’ll start your brief workflow from the next cycle.
```

## 10. Follow-Up If They Go Silent

```text
Just following up on FinStack Brief.

If you still want a sample, send me:
- watchlist
- format
- use case

I can prepare one quickly.
```
