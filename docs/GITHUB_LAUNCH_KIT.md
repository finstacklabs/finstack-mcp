# GitHub Launch Kit

## Repo description

Open-source MCP server for Indian markets, stock research, NIFTY/BANKNIFTY F&O setups, and AI-native trading workflows.

## One-line hook

Ask Claude one market question and get debate, sentiment, smart money, ranking, and F&O setup in one stack.

## Short pitch

FinStack MCP is a market-intelligence layer for Indian equities and index options that works inside Claude, Cursor, and other MCP clients. It combines live data, AI debate, smart-money signals, research workflows, and morning briefs in one open-source package.

## Why this wins

- Indian markets first
- MCP-native instead of dashboard-only
- real daily hooks like `get_morning_fno_brief`
- differentiated tools like stock debate, stock timeline, Telegram tracker, and GST-to-stock context

## FinStack MCP v0.10.0

This release turns FinStack into a stronger daily-use workflow, not just a market-data toolbox.

### New
- `get_fno_trade_setup` for NIFTY / BANKNIFTY options calls
- `get_morning_fno_brief` for 8:15 AM F&O summaries
- research and ranking tools for watchlists, timelines, and signal scoring
- clearer README positioning and demo assets

### Why it matters
- ask one market question in Claude or Cursor
- get structured signals with factors and risks
- use FinStack for daily triage, stock research, and morning market prep

### Install
```bash
pip install finstack-mcp
```

or

```bash
uvx finstack-mcp
```
```
