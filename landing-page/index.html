why we built this

Built different.
On purpose.

01
SEBI mandates it. They charge for it.

ESG/BRSR data is legally required to be publicly disclosed. We just make it queryable — no ₹18k/yr subscription, no data desk, no login wall.

02
Real-time. Not 15-minute delayed.

Most free tools give you data that's 15 minutes old. Via Angel One broker integration, finstack-mcp gives you live quotes — same as paid terminals.

03
58 tools. One pip install.

No API keys to manage. No webhooks to configure. No JSON to parse. Just install and ask Claude. Works out of the box in 60 seconds.
